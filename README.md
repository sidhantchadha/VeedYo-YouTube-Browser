# VeedYo - YouTube video browser

VeedYo is a YouTube video browsing app for iOS devices with iOS 11 card-like animation support. Language: Swift 4.0 IDE: Xcode 9.4.1

Frameworks:

1.Firebase
2.GoogleSignIn
3.Hero Transitions
